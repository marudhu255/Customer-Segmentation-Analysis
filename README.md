**Customer Segmentation and Purchase Behavior Analysis

Overview

This project focuses on analyzing customer purchasing behavior using Power BI and the Online Retail dataset. The dashboard provides insights into customer segments, revenue trends, purchase frequency, and country-wise sales performance through interactive visualizations and KPI metrics.

Objectives

Analyze customer purchasing behavior

Perform customer segmentation

Identify high-value and low-value customers

Visualize revenue and sales trends

Generate business insights for decision-making


Tools & Technologies

Microsoft Power BI

Power Query

DAX (Data Analysis Expressions)

Excel / Online Retail Dataset


Dataset

The project uses the Online Retail dataset containing:

Customer ID

Invoice Number

Product Description

Quantity

Unit Price

Country

Invoice Date


Data Cleaning

The following preprocessing steps were performed:

Removed null values

Removed duplicate records

Filtered negative quantities

Created calculated columns and measures

Transformed data types


KPI Metrics

Total Revenue

Total Customers

Purchase Frequency

Customer Segmentation


Dashboard Features

Customer Segmentation Analysis

Interactive Charts and KPI Cards
